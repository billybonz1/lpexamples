Change Color

Check <head> part of the index.html. You will find all colors link available. blue color is default. So if you want to use any other color you can remove the comment make other color active.

<!-- <link rel="stylesheet" href="css/colors/red.css"> -->

<!-- <link rel="stylesheet" href="css/colors/green.css"> -->

<!-- <link rel="stylesheet" href="css/colors/yellow.css"> -->

Contact Form

Contact form will use php to send emails. So just open sendmail.php file. Change the current email lesh1j@yandex.by to your email address. Your form is ready.

* Contact form require web server to send email. Emails can go to your spam/junk folder.
* If you use custom email like yourname@yourcompany.com then may be your mail client can block php mail completely. I recommend to use gmail or athor popular email address. 
* If you want perfect php mail system. please use https://github.com/PHPMailer/PHPMailer
